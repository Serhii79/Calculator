# Calculator

This app implements a simple calculator with basic arithmetic operations. It can perform addition, subtraction, multiplication, and division.

## Preconditions.

Maven Compiler Source 1.8
JUnit 4.13.2
Maven Surefire Plugins 3.2.5
Installed locally apache-maven-3.9.6

## Installation

1. Download the code from the repository.
2. Open the project in your integrated development environment (IDE).

## Passing tests

1. Open the Terminal.
2. Enter at the command prompt: mvn test.
3. Click the "Enter" button to get the result.

## Checking the results

Expected result:
Tests run: 5, Failures: 0, Errors: 0, Skipped: 0
